# So geht's

1. Öffne **Visual Studio Code** und erstelle ein leeres Projekt
2. Öffne die **Claude Code Erweiterung** in der Seitenleiste
3. Kopiere diese Zeile in den Chat:

```
https://github.com/dasboe/ai-learn/blob/main/bootstrap.md
```

Claude richtet alles ein. Danach:

1. Schliesse diese Claude-Instanz
2. Öffne Claude Code neu im selben Ordner
3. Tippe `/start`

Jede Session: Claude Code öffnen, `/start`, lernen, `/end`, schliessen.
